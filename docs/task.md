# Техническое задание

## 1. Описание задачи
Разработать утилиту, которая принимает текст (через аргумент командной строки или stdin) и определяет:
- язык текста
- уровень уверенности
- список слов (токены)

Утилита должна использовать библиотеку `polyglot`.

## 2. Цель работы
Освоить базовые возможности `polyglot`: определение языка и токенизацию на разных языках.

## 3. Функциональные требования
- Поддержка текста на русском, английском, французском, арабском, китайском
- Вывод: название языка, код (например, `ru`), уверенность (%), список слов
- Обработка коротких текстов (от 3 слов)
- Корректная работа с Unicode (включая эмодзи и спецсимволы)

## 4. Нефункциональные требования
- Код должен быть читаемым
- Использовать только `polyglot` и стандартную библиотеку
- Не использовать сторонние зависимости для определения языка

## 5. Ограничения и допущения
- Предполагается, что `polyglot` и его модели установлены
- Текст подаётся как одна строка 
- Утилита запускается в среде с поддержкой UTF-8

## 6. Источник ТЗ
Самостоятельная постановка.
