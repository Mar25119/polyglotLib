# Выводы по работе

## 1. Что было изучено
Я освоил базовые функции `polyglot`: определение языка (`Detector`) и токенизацию (`Text.words`). Убедился в его способности работать с разными письменностями — от кириллицы до иероглифов.

## 2. С какими трудностями столкнулся
Главная сложность — установка. На Windows без WSL почти невозможна. Также пришлось разобраться с тем, что короткие тексты дают низкую уверенность (например, "Hi" → 60%).

## 3. Что получилось удачно
Утилита корректно обрабатывает многоязычные тексты и выдаёт структурированный результат. Особенно впечатляет работа с китайским и арабским без пробелов.

## 4. Где инструмент уместен
- Академические проекты по лингвистике
- Анализ пользовательского контента в международных сервисах
- Образовательные демонстрации многоязычности

## 5. Где инструмент неуместен
- Production-сервисы с требованиями к скорости и поддержке
- Проекты, где нужна только пара языков 
- Системы с ограничениями на память

## 6. Итоговая оценка
Оценка: **7/10**  
`polyglot` — уникальный, но устаревающий инструмент. Отличен для обучения и исследований, но требует осторожности в реальных системах.
