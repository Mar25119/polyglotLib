# Проект: Многоязычный анализ текста с polyglot

Учебный проект, демонстрирующий возможности библиотеки `polyglot` для определения языка и токенизации текста на **более чем 190 языках**, включая русский, китайский, арабский и другие.

## Задача
Автоматически определить язык введённого текста и разбить его на слова с учётом правил конкретного языка.

## Требования

- Python 3.7+
- Библиотека `polyglot`
